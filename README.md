# Mini_organiser
Мини органайзер с калькулятором, таймером и телефонной книгой.
Школьный проект ученика 9 класса

## 🛠️ Стек технологий

- Python 3
- Tkinter — интерфейс
- SQLite — база данных

## 🚀 Запуск

```bash
git clone https://github.com/GIROSKOP/Mini_organiser.git
cd Mini_organiser-main
python organiser.py
```
